1.1 HTML 结构

我们首先创建一个基础的 HTML 页面，其中包含一个模态框用于放大图片显示，以及几个示例图片：

<!DOCTYPE html>
<html lang="zh">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>透明背景放大图片效果</title>
  <style>
    /* 透明背景模态框 */
    .modal {
      display: none;
      position: fixed;
      z-index: 1000;
      left: 0;
      top: 0;
      width: 100%;
      height: 100%;
      overflow: auto;
      background-color: rgba(0, 0, 0, 0.7);
      text-align: center;
      padding-top: 60px;
    }
    /* 放大的图片 */
    .modal-content {
      margin: auto;
      display: block;
      width: 80%;
      max-width: 700px;
      transition: transform 0.25s ease;
    }
    /* 关闭按钮 */
    .close {
      position: absolute;
      top: 15px;
      right: 35px;
      color: #fff;
      font-size: 40px;
      font-weight: bold;
      cursor: pointer;
      background-color: transparent;
      border: none;
    }
    .close:hover {
      color: #f1f1f1;
    }
  </style>
</head>
<body>

  <!-- 模态框 -->
  <div class="modal" id="myModal" onclick="closeModal(event)">
    <button class="close" onclick="closeModal(event)">&times;</button>
    <img class="modal-content" id="img01">
  </div>

  <h2>点击图片放大并透明背景示例</h2>

  <!-- 示例图片 -->
  <img src="https://via.placeholder.com/150/0000FF/808080?Text=Demo1" width="150" height="150" onclick="openModal(this)">
  <img src="https://via.placeholder.com/150/FF6347/FFFFFF?Text=Demo2" width="150" height="150" onclick="openModal(this)">
  <img src="https://via.placeholder.com/150/000000/FFFFFF?Text=Demo3" width="150" height="150" onclick="openModal(this)">

  <script>
    function openModal(imgElement) {
      var modal = document.getElementById("myModal");
      var modalImg = document.getElementById("img01");
      modal.style.display = "block";
      modalImg.src = imgElement.src;
    }
    function closeModal(event) {
      if (event.target === event.currentTarget || event.target.className.indexOf('close') !== -1) {
        document.getElementById("myModal").style.display = "none";
      }
    }
  </script>
</body>
</html>

1.2 代码说明

  • 模态框：通过一个 <div> 实现，设置 display: none; 隐藏，当点击图片时通过 JavaScript 将其显示出来。
  • 点击图片放大：给图片添加 onclick="openModal(this)" 事件，当点击时，将图片的 src 赋值给模态框中的 <img> 元素并显示模态框。
  • 关闭模态框：点击模态框背景或关闭按钮时，调用 closeModal(event) 函数，将模态框隐藏。
